# Terminal

Second cerveau personnel — pas un cours, pas un Notion, pas un journal. Un endroit pour retrouver n'importe quelle connaissance ou erreur en moins de 5 secondes.

Le trading est le premier module. L'architecture (concepts, erreurs, stratégies, journal, ressources) est pensée pour accueillir d'autres domaines plus tard.

## Principes

- **Mobile first** — tout utilisable au pouce.
- **Local first** — aucune base de données distante, tout vit dans IndexedDB, export/import JSON.
- **Minimalisme** — pas de dashboard, pas de widgets, pas de stats inutiles.

## Pages

Accueil, Erreurs, Concepts, Stratégies, Journal, Ressources.

## Stack

React, Vite, TypeScript, Tailwind CSS v4, React Router, Zustand, Dexie (IndexedDB), Fuse.js.

## Développement

```bash
npm install
npm run dev
```
